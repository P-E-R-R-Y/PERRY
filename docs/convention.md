# Convention

## Convention C++

- **FOLDER** -> snake_case
- **FILE**
  - **CLASSE** -> PascalCase
  - **UTILITY** -> snake_case
- **NAMESPACE** -> snake_case
- **CLASSE & STRUCT** -> PascalCase
- **FONCTION** -> camelCase
- **VARIABLE** -> snake_case
- **CONSTANT & ENUM** -> UPPER_SNAKE_CASE

## Convention git

(see [conventional_commit](https://www.conventionalcommits.org/en/v1.0.0/))

- **MESSAGE** -> `<type>(<scope>): <gitmoji> what you've done so far.`
